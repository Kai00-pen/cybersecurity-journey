# Linux Commands

These are basic Linux commands I'm learning in my cybersecurity class.

---

## Navigation Commands

pwd  
- Shows current directory (where you are)

ls  
- Lists files and folders

ls -l  
- Shows detailed list (permissions, size, etc.)

cd  
- Changes directory (moves you into a folder)

cd ..  
- Goes back one folder

---

## File Commands

cat filename  
- Displays contents of a file

head filename  
- Shows the top of a file

tail filename  
- Shows the bottom of a file

touch filename  
- Creates a new file

cp file1 file2  
- Copies a file

mv file1 file2  
- Moves or renames a file

rm filename  
- Deletes a file

---

## Process Commands

ps  
- Shows running processes

top  
- Shows live processes (real-time)

kill PID  
- Stops a process using its ID

pkill name  
- Kills a process by name

---

## System Commands

clear  
- Clears the terminal

history  
- Shows previous commands

echo text  
- Prints text to the terminal

---

## Help Commands

man command  
- Opens manual (help page)

help command  
- Shows help for built-in commands
